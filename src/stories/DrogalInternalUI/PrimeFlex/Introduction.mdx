# Sobre

PrimeFlex é leve, responsivo e totalmente compatível com os componentes PrimeReact. Ele fornece um conjunto de classes utilitárias que facilitam a criação de interfaces sem a necessidade de escrever CSS customizado.

## Principais Recursos

- **Display** - Classes para controlar o tipo de exibição dos elementos
- **Flexbox** - Sistema completo para layouts flexíveis
- **Grid** - Sistema de grid de 12 colunas responsivo
- **Spacing** - Utilitários de margin e padding
- **Typography** - Classes para texto e tipografia
- **Colors** - Sistema de cores integrado
- **Elevation** - Sombras e profundidade
- **Border** - Classes para bordas e arredondamento
